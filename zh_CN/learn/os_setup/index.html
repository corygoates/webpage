A beginner's tutorial on the software that needs to be installed to start with Fortran development


Authors: Stavros Meskos, Laurence Kedward, Arjen Markus, Ondřej Čertík, Milan Curcic

Last update: 16-Feb-2021

设置操作系统#

在这本迷你书中，我们解决了许多新 Fortran 程序员遇到的第一个问题。在一切之前，你需要选择一个编译器并安装它。然后，你将需要一个文本编辑器，或者可能需要一个 IDE（集成开发环境）。有很多选项，其中大多数在所有主要操作系统（操作系统）中都可用。但是，安装和配置它们的过程在 Windows、Linux 和 macOS 之间存在很大差异。有几个 Fortran 编译器。在这里，我们挑选符合特定标准的那些，并提供有关如何在所有上述操作系统中安装它们的指南。还提供了流行的文本编辑器和 IDE 列表以及有关如何在它们之间进行选择的信息。

备注

如果你已经设置了环境，你可以跳过本教程并继续阅读快速入门 Fortran 教程以开始编写你的第一个 Hello World 程序或阅读构建程序迷你书以了解编译器的工作原理。

文本编辑器和IDE的选择在开发者中一直存在争议。你选择什么是个人品味的问题。每个解决方案都有自己的优点和缺点。因此，在本指南中，我们不会将读者推向一个或另一个方向。但是，我们将列出最受欢迎的工具。
