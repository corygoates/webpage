IDE (Интегрированная среда разработки)#

IDE (Интегрированная среда разработки) – полная среда разработки программного обеспечения, в которой все те возможности, которые поставляются в текстовых редакторах в виде внешних дополнений, уже встроены. IDE обычно оптимизирована под определённый набор языков программирования. Например, обычным для IDE является позиционирование себя как продукта предназначенного для компилируемых или интерпретируемых языков, или даже для одного языка, или в зависимости от направления разработки – предназначенного для научной или web разработки. IDE рекомендуются для начинающих программистов, так как в них можно сразу начать писать код, затратив минимальные усилия по настройке после установки. Однако в профессиональной среде, как отдельные разработчики, так и команды разработчиков, довольно часто выбирают IDE для больших проектов из-за некоторых уникальных возможностей, которые они могут предложить по сравнению с текстовым редактором.

Список популярных IDE, которые обеспечивают хорошую поддержку языка Fortran, приведённый в алфавитном порядке:

  • AbsoftTools: коммерческий кросс-платформенный продукт, поставляется с собственным компилятором языка Fortran от Absoft.

  • Code::Blocks: бесплатный кросс-платформенный продукт, поддерживает множество компиляторов. В интернете можно можно найти множество руководств по его установке и добавлению компилятора Fortran. Также здесь доступна недавняя презентация его возможностей по поддержке языка Fortran и планов по их расширению, представленная на Международной конференции по Фортрану 2020 (International Fortran Conference 2020) одним из его основных разработчиков.

  • Eclipse-Photran Photran – IDE и инструмент для рефакторинга для языка Fortran на основе Eclipse и C/C++ Development Tools. Бесплатный кросс-платформенный продукт с поддержкой множества компиляторов.

  • Geany: бесплатный кросс-платформенный продукт с поддержкой множества компиляторов.

  • NAG Fortran Builder – коммерческий продукт, доступный для Windows и MacOS, поддерживает собственный Fortran компилятор NAG.

  • Plato – коммерческий кроссплатформенный продукт, поставляется с собственным Fortran компилятором FTN95 от Silverfrost.

  • SimplyFortran – коммерческий кросс-платформенный продукт с поддержкой компилятора GNU Fortran (gfortran).

  • Visual Studio – его бесплатная версия Community Edition 2019 доступна для Windows и macOS и поддерживает только компилятор Intel’s Fortran, который теперь поставляется со многими другими инструментами для высокопроизводительных вычислений (HPC) в рамках продукта Intel oneAPI.
